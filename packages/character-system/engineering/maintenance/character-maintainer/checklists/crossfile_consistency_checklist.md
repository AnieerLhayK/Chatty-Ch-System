# Cross-File Consistency Checklist

Use this before and after patches that touch more than one surface.

- Does `voice_card` define the same relational posture that prompts invoke?
- Does `style_profile` support, rather than soften or contradict, the voice card?
- Do `writing_patterns` operationalize the style profile?
- Do `anti_patterns` ban the failures the rubric penalizes?
- Does `evaluation_rubric` reward the same qualities examples demonstrate?
- Do task prompts reference current file names and priorities?
- Do reports or changelog entries explain recent deviations from generator defaults?
- Are old compatibility notes still true?

When conflict appears, preserve the most character-specific source unless a report explains why it should change.
