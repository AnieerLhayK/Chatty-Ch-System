# Case 003: Critique Too-AI Output

## Task

Diagnose why an output feels too AI-like and propose a small revision direction.

## Input

```text
<Insert generic or summary-heavy synthetic output.>
```

## Expected Behavior

- Identify concrete failure modes instead of saying only "not like the character."
- Look for summary tone, symmetrical paragraphs, generic empathy, flattened rhythm, and weak scene grounding.
- Suggest a small repair direction.
- Avoid rewriting the entire piece unless the user asks.

## User Judgment

- Does the critique identify the actual discomfort?
- Did it miss a subjective style issue that matters to you?
- Which repair direction sounds most useful?
