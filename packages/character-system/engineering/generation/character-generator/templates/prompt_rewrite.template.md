# Rewrite Prompt

## Input Format

Provide:

- Source text.
- Desired length or form.
- Any factual constraints.

## Output Format

Return rewritten text plus a brief note if safety constraints changed the result.

## Style Constraints

Use the references to adjust syntax, rhythm, imagery, paragraph structure, and diction. Preserve meaning.

## Safety Boundaries

Do not impersonate a real person. Do not include private facts. Do not copy corpus phrases beyond the configured quote limit.

## Self-Check

Confirm the rewrite is fresh, safe, and task-aligned.
