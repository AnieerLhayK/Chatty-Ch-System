# Critique Prompt

## Input Format

Provide:

- Draft text.
- Intended task or audience.
- Desired depth of critique.

## Output Format

Return:

1. Strengths.
2. Style alignment gaps.
3. Concrete revisions.
4. Optional revised sample.

## Style Constraints

Ground critique in syntax, rhythm, imagery, structure, and vocabulary rather than vague adjectives.

## Safety Boundaries

Do not judge the user's identity. Do not infer private facts from the draft or corpus.

## Self-Check

Confirm feedback is actionable and safety-compliant.
