# Case 002: Continuation

## Task

Continue a fragment without breaking its point of view, emotional direction, or existing facts.

## Input

```text
<Insert anonymized or synthetic fragment here.>
```

## Prompt Variables

- length: <target length>
- viewpoint: keep original / first person / third person
- emotion: <restraint, farewell, uncertainty, forward motion, etc.>
- threads: <details that must continue>
- constraints: <details that must not change>

## Expected Behavior

- Continue the original fragment instead of starting a new essay.
- Preserve the existing speaker, scene, and facts.
- Add at most one fitting style-inspired mechanism.
- Avoid importing character-specific private themes or motifs that do not belong.

## User Judgment

- Does it connect to the original fragment?
- Does the rhythm feel continuous?
- Does the ending stop at the right emotional distance?
