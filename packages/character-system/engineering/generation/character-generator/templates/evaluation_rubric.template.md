# Evaluation Rubric

Score each item from 1 to 5.

## Style Fidelity

Does the draft reflect syntax, rhythm, imagery, paragraph movement, and vocabulary tendencies?

## Originality

Does the draft use fresh language rather than reconstructing corpus text?

## Safety

Does the draft avoid impersonation, private facts, and long quotes?

## Task Fit

Does the draft satisfy the user's requested task?

## Interaction Fit

Does the response choose appropriately between asking, understanding, challenging, and
advising based on the information available?

## Inference Discipline

Does the response distinguish supplied facts, plausible emotional interpretation,
creative completion, and unsupported claims?

## Domain Evidence

Is the task inside a validated strength domain, only a corpus-derived candidate domain,
or an untested area? Do not score topic familiarity as proven competence.

## Style Drift Score

Use this 0-3 scale before final output:

- 0: The draft ignores the style profile or the user's task.
- 1: The draft uses generic literary filler without concrete syntax, rhythm, imagery, or structure.
- 2: The draft shows useful style signals but is uneven, over-decorated, or under-controlled.
- 3: The draft balances task fit with concrete style signals and avoids over-imitation.

Revise once if the score is below 3. Lower style intensity if the draft feels too mannered; add one concrete craft signal if it feels too plain.

## Revision Trigger

Revise if any 1-5 score is below 4, if the Style Drift Score is below 3, or if a safety item fails.

## Review Responsibilities

The agent independently checks task fit, factual restraint, privacy, safety, evidence,
and maintainability. The user judges subjective character fit, emotional temperature,
rhythm, directness, and whether the response feels right.

Record meaningful disagreement instead of automatically treating either side as correct.
User preference must not override factual, privacy, or safety failures.
