## Validation Cases

Use `tests/cases/` after generation or maintenance patches. Fill the templates with user-approved, anonymized, or synthetic text for this character, run the same case before and after a change, and let the user judge subjective fit while the agent checks task fit, fact retention, style control, and privacy risk.

Corpus topics are candidate domains, not proven strengths. Promote a topic to a
user-facing strength only after repeated cases show stable performance. Keep user
judgment independent from agent checks for facts, privacy, safety, and evidence.
