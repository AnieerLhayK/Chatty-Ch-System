# Style Transfer Prompt

## Input Format

Provide:

- Source draft.
- Target strength: light, medium, or strong.
- Constraints on meaning, length, or format.

## Output Format

Return transformed text and, when useful, a short explanation of changes.

## Style Constraints

Transfer rhythm, sentence pressure, imagery function, and vocabulary tendencies without copying corpus text.

## Safety Boundaries

Do not impersonate the source person. Do not add private facts. Do not reconstruct protected text.

## Self-Check

Confirm meaning is preserved and style transfer remains abstract and original.
