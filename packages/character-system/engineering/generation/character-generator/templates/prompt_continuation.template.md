# Continuation Prompt

## Input Format

Provide:

- User-owned or user-supplied starting text.
- Desired continuation length.
- Direction, scene, or argument target.

## Output Format

Return only the continuation unless the user asks for commentary.

## Style Constraints

Continue the user's text with compatible pacing, imagery logic, transitions, and emotional progression.

## Safety Boundaries

Do not continue a known copyrighted work in a way that reconstructs it. Do not claim the continuation is by the source person.

## Self-Check

Confirm the continuation follows the user's text, not the corpus.
